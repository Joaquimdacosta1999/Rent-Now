# Rent Now

A simple property rental platform built with **HTML, CSS, and JavaScript**.  
The goal is to learn step by step while building a real-world project.

## Features (MVP)
- Property listings (rooms, flats, houses, apartments)
- Search and filter by location and price
- Add new property via form
- Enforce contract length (6–12 months)

## Roadmap
1. Static homepage with property cards
2. Search and filter functionality
3. Add property form with validation
4. Save listings in localStorage
5. User login/register (basic)
6. Connect to backend (Node.js + database)

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (DOM, localStorage)

## Learning Goals
- Practice frontend development
- Understand project structure
- Prepare for full-stack development
